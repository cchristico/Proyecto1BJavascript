Strings .includes() .repeat()

Las opciones de la libreria estandar de ES6 ha incrementado y, dentro de esos cambios se encuentran los nuevos metodos de string, como .include() y .repeat(). 

var str = 'Hola mundo, frase ser o no ser, esa es la pregunta.';

//.include(Substring a buscar)
console.log(str.includes('Hola')); // true - consta en la cadena
console.log(str.includes('es la pregunta')); // true - consta en la cadena
console.log(str.includes('hola Mundo')); // false - Compara mayusculas y minusculas
console.log(str.includes('SER', 1)); // false - Compara mayusculas y minusculas
console.log(str.includes('Pregunta')); // false - Compara mayusculas y minusculas
