Template literals

El uso de Template Literals o plantillas literales presenta caracteristicas asociadas a la construccion de strings sin el uso de caracteres de escape o ayuda 

//Impresion de Comillas en Strings
var text = "Expresiones \"con comillas\" a ser impresas en consola."; // Permite imprimir comillas
console.log(text);

let text2 = `Esta expresion tiene "doble comillas".`; // Permite imprimir comillas
        console.log(text2);
        
//Concatenacion
var name = 'Max'; 
var age = 8;

//Antes -- Concatenacion con '+'
console.log('Mi perro se llama ' + name + ' y tiene ' + age + ' anios.');

const name2 = 'Misty';
const age2 = 6;

// Concatenacion con ${variable}
console.log(`Mi gata se llama ${name2} y tiene ${age2} anios.`);

//Versiones anteriores de ES para salto de linea
var text3 = (
    'cat\n' +
    'dog\n' +
    'nickelodeon'
);
console.log(text3);

//Se mantiene los saltos de linea que se especifica
let text4 = (`cat
            dog
            nickelodeon`);
console.log(text4);
