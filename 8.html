Destructuring objects

Destructuring objects es una manera conveniente de extraer múltiples valores de datos almacenados en los objetos (posiblemente anidados). 

const obj = { first: 'Jane', last: 'Doe' };
const {first: f, last: l} = obj;
    // f = 'Jane'; l = 'Doe'

// {prop} is short for {prop: prop}
const {first, last} = obj;
    // first = 'Jane'; last = 'Doe'

Destructuring objects yuda con valores de retorno de procesamiento: 

 
const obj = { foo: 123 };

const {writable, configurable} =
    Object.getOwnPropertyDescriptor(obj, 'foo');

console.log(writable, configurable); // true true
