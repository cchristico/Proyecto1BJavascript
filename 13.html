Promises (goodbye callback hell)

Una promesa o promise se usa para computacion asincrona. Puede estar definida en tres estados: pendiente, cumplida o rechazada, dependiendo si la operacion aun no se realiza, se realizo satisfactoriamente o fue fallida, respectivamente 

// Método de devolución de llamada
async1(function(){
    async2(function(){
        async3(function(){
            ....
        });
    });
});

// Enfoque de la promesa
var task1 = async1();
var task2 = task1.then(async2);
var task3 = task2.then(async3);

task3.catch(function(){
    // Resuelve tus errores lanzados desde task1, task2, task3 aquí
})

// Enfoque de la promesa con encadenamiento
async1(function(){..})
    .then(async2)
    .then(async3)
    .catch(function(){
        // Resuelve tus errores lanzados aquí
    })
